# Network-Intrusion_Dataset-CIC-IDS--2017
This project presents a multi-class Network Intrusion Detection System (NIDS) trained to differentiate between twelve types of network traffic. One of the types of traffic is categorized as normal network traffic, while the rest are distinct network attack types.

The interactive dashboard for this project is publicly accessible at:  
https://multiclassnidscicids2017.streamlit.app/
